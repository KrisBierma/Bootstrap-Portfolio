node liri.js

If no command is given, the user is given acceptable commands.

Command: my-tweets

Displays a list of 20 tweets.

Write Info to Text File

All displayed info is also writted to the log.txt file.

Command: spotify-this (no title)

Returns default info for the song "The Sign."

Command: spotify-this (with a title)

Display's the song title, album, artist, and preview link from Spotify.

Command: movie-this (no movie title)

Returns info for default movie "Mr. Nobody."

Command: movie-this (with a movie title)

Displays movie info from OMDB.

Command: do-what-it-says

Reads information from the random.txt file, which includes a song title, and consoles the Spotify information for that song.
